# Smart-Device-Usage-Analysis

Bellabeat Case Study — SQL Analysis

This repository contains a comprehensive data analysis of **smart device usage patterns**. The goal of this project is to analyze how consumers use their non-medical smart devices to provide high-level recommendations for marketing strategies.

## 📖 Project Overview
This analysis follows the Google Data Analytics Professional Certificate methodology (**Ask, Prepare, Process, Analyze, Share, and Act**) to uncover trends in physical activity, sleep patterns, and daily usage.

[![Kaggle](https://img.shields.io/badge/Kaggle-View_Notebook-00BBFF?style=for-the-badge&logo=kaggle)](https://www.kaggle.com/code/joylorna/smart-device-usage-analysis)

**Key Insights & Analysis:**
* **Activity Levels:** Deep dive into sedentary vs. active minutes and their impact on user health.
* **Sleep Correlation:** Analyzing the relationship between daily steps and sleep quality.
* **Usage Trends:** Identifying which days of the week users are most engaged with their devices.
* **Marketing Recommendations:** Data-driven suggestions for product positioning based on user behavior.

## 🛠️ Tech Stack
* **Data Extraction & Querying:** SQL
* **Data Analysis & Logic:** Microsoft Excel
* **Visualization:** Microsoft Excel

## 📊 Data Source
The analysis utilizes the **FitBit Fitness Tracker Data** (available on Kaggle), which contains personal fitness tracker data from thirty fitbit users.
