# Smart-Device-Usage-Analysis
Bellabeat Case Study — SQL Analysis

This case study analyzes user activity and sleep behavior using the Bellabeat Smart Devices Health Dataset. The objective is to identify trends in physical activity, sleep duration, and engagement patterns, and to provide actionable recommendations that can help Bellabeat improve user wellness and strengthen product adoption. The analysis bridges technical SQL workflows with business insights, demonstrating how wearable data can inform strategic decision-making.
